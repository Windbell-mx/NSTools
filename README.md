# 🛡️ 渗透测试信息收集平台

自动化信息收集平台，支持域名枚举、子域名发现、端口扫描、目录探测、指纹识别等功能。

## 📋 功能特性

- **子域名扫描**: 自动发现目标域名的子域名
- **端口扫描**: 检测目标IP的开放端口
- **目录扫描**: 探测Web目录和敏感文件
- **指纹识别**: 识别Web服务器和技术栈
- **DNS收集**: 获取DNS记录信息
- **报告生成**: 一键导出扫描报告（Markdown + CSV）

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run app.py
```

### 3. 访问界面

打开浏览器访问 http://localhost:8501

## 📁 项目结构

```
pentest-collector/
├── app.py                    # Streamlit主应用
├── requirements.txt          # 依赖列表
├── modules/
│   ├── scanner.py            # 扫描器模块
│   └── report_generator.py   # 报告生成器模块
└── reports/                  # 生成的报告目录
```

## ⚙️ 环境要求

- Python 3.8+
- Streamlit 1.32+
- 其他依赖见 requirements.txt

## 📝 使用说明

1. **创建任务**: 在左侧菜单选择「创建任务」，输入目标域名/IP
2. **选择模块**: 选择要执行的扫描模块（DNS、子域名、端口、目录、指纹）
3. **开始扫描**: 点击「开始扫描」按钮
4. **查看结果**: 扫描完成后，在「任务详情」中查看结果
5. **导出报告**: 支持下载CSV数据和完整Markdown报告
