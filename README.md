# 🛡️ 渗透测试信息收集平台

自动化信息收集平台，支持域名枚举、子域名发现、端口扫描、目录探测、指纹识别等功能，集成多种网络空间测绘平台。

## 📋 功能特性

### 🌐 信息收集
- **DNS收集**: 获取A、AAAA、MX、NS、TXT等DNS记录
- **WHOIS信息**: 获取域名注册商、注册人、联系方式等
- **子域名扫描**: 自动发现目标域名的子域名（集成subfinder、amass工具）

### 🔎 服务扫描
- **端口扫描**: 检测目标IP的开放端口和服务类型
- **指纹识别**: 识别Web服务器、开发框架、CMS等技术栈

### 📁 Web探测
- **目录扫描**: 探测Web目录和常见路径
- **敏感文件探测**: 检测.env、.git、备份文件等敏感路径

### 🛡️ 安全检测
- **SSL证书检测**: 获取证书颁发机构、有效期、加密算法等
- **WAF识别**: 识别Cloudflare、阿里云WAF、腾讯云WAF等
- **CDN识别**: 识别Cloudflare、阿里云CDN、腾讯云CDN等
- **云服务识别**: 识别AWS、Azure、GCP、阿里云、腾讯云等
- **ICP备案查询**: 查询域名备案信息

### 🔭 测绘平台集成
- **FOFA**: 网络空间测绘系统
- **Shodan**: 搜索引擎
- **Hunter**: 网络空间搜索引擎
- **ZoomEye**: 网络空间搜索引擎

### 📊 数据处理流程
- **数据收集**: 多源数据采集（DNS、WHOIS、子域名、端口等）
- **数据验证**: 验证域名格式、IP格式、服务可用性
- **数据清洗**: 去重处理，避免重复结果
- **结果展示**: 表格形式展示，支持状态图标

### 📈 实时进度展示
- **真实进度更新**: 每个模块执行完成后更新进度
- **阶段状态指示**: 当前执行阶段和详细信息
- **状态图标**: ✅已完成 / 🔄进行中 / ⏳等待中
- **自动刷新**: 扫描过程中页面自动更新

### 📄 报告生成
- 一键导出扫描报告（Markdown格式）
- 支持CSV数据导出

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 安装子域名工具（可选）

```bash
# 安装subfinder
go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest

# 安装amass
go install github.com/OWASP/Amass/v4/...@latest
```

### 3. 启动应用

```bash
streamlit run app.py
```

### 4. 访问界面

打开浏览器访问 http://localhost:8501

## 📝 使用说明

### 基础使用

1. **创建任务**: 在左侧菜单选择「创建任务」，输入目标域名/IP
2. **选择模块**: 选择要执行的扫描模块
3. **开始扫描**: 点击「开始扫描」按钮
4. **查看进度**: 在「任务详情」中查看实时扫描进度
5. **查看结果**: 扫描完成后，查看表格形式的扫描结果
6. **导出报告**: 支持下载CSV数据和完整Markdown报告

### 配置测绘平台

1. 在左侧菜单选择「系统设置」
2. 选择要启用的测绘平台（FOFA、Shodan、Hunter、ZoomEye）
3. 输入API密钥或账号信息
4. 点击「保存配置」

### 使用测绘搜索

1. 在「仪表盘」页面的测绘搜索区域输入关键词
2. 点击「搜索」即可查询所有启用的平台

## 🎯 子域名工具说明

| 工具 | 安装方式 | 说明 |
|------|----------|------|
| subfinder | `go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest` | 被动子域名枚举工具 |
| amass | `go install github.com/OWASP/Amass/v4/...@latest` | 主动+被动子域名枚举工具 |

> **注意**: 如果未安装这些工具，系统会自动降级到基础枚举方法，不会报错。

## 📊 数据处理流程

```
数据收集 → 数据验证 → 数据清洗 → 结果展示
    ↓           ↓           ↓           ↓
 DNS/WHOIS   格式验证     去重处理     表格展示
 子域名/端口  可用性检测   标准化处理   统计摘要
```
